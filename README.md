# c64emu

C64 emulator for the browser; written in TypeScript.

NOTE: THIS PROJECT IS WORK IN PROGRESS!

Visit https://c64emu.github.io/

# Installation

Runtime:

-   Only a HTTP browser is needed

Development dependencies:

-   node.js
-   TypeScript

Build:

-   npm install
-   npm run build
